KDE AI Chat (v1.3.1)
A native, fast, and easy-to-use AI chat assistant widget for the KDE Plasma 6 desktop. Connect to your favorite AI providers, drag and drop files, schedule automated tasks, and securely save your API keys.

KEY FEATURES:

1. OpenCode Developer Bridge
-> Link your chat directly to your local code workspace. You can run code, search the web, and execute developer workflows right from your desktop panel. Basically running opencode through this widget.

2. Local Voice Mode
-> Talk to the AI hands-free. Includes real-time speech-to-text input and spoken audio responses using local python-based voice tools. A global stop button allows you to interrupt speech playback instantly.

3. Drag and Drop Files
-> Paste or drag images, PDFs, CSVs, Word documents, and text files directly into the chat. You can even send files by themselves without typing a question.

4. Prompt Templates
-> Save prompts you use frequently in the settings. Type a slash followed by your template name in the chat box to insert them instantly.

5. Task Scheduler
-> Set up recurring tasks using simple calendar schedules or cron expressions. Great for running automated code checks or asking periodic questions in the background.

6. Edit Messages and Rewind
-> Edit any older question in the thread to automatically clear subsequent history and restart the conversation from that point.

7. Secure Key Storage
-> Safely save your API keys. The widget encrypts keys in KDE's secure password manager (KWallet) and automatically falls back to a local configuration file (~/.config/kdeaichatrc) if KWallet is not running.

8. 15+ AI Providers
-> Works out of the box with OpenAI, Claude, Groq, DeepSeek, Gemini, OpenRouter, Mistral, Ollama, LM Studio, LiteLLM Proxy, and more. For other custom providers not listed, you can configure them in OpenCode and access them directly through OpenCode mode.

9. Export Chats
-> Save your chat history as Markdown (.md) or plain text (.txt) files with a single click.

10. Quick Navigation
-> Easily jump between your questions in a long chat using the Up and Down arrow buttons on the toolbar.

11. Smooth Scrolling
-> Enhanced chat bubble layout scrolling that avoids jittering or jumping when responses load.

12. Resizable Layout
-> Drag the bottom-right corner of the chat popup to resize it exactly how you want.


WHAT'S NEW IN VERSION 1.3.1:

-> Fixed voice background services autostarting on PC reboot when voice tools are disabled in settings.
-> Automatically disables systemd user units (`kde-ai-stt.service` and `kde-ai-tts.service`) when voice tools or TTS are turned off.
-> Widget startup automatically checks and syncs systemd service state to avoid unnecessary background RAM usage.
