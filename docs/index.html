Installation & Setup

1. Quick Install

Clone the repository and run the installation script. Requires KDE Plasma 6 and Qt 6.

git clone https://github.com/racstan/rachitkdeaichat
cd rachitkdeaichat
./install.sh
systemctl --user restart plasma-plasmashell.service

2. Panel Integration

  1. Right-click your Plasma panel and select Add Widgets…
  2. Search for Kai Chat.
  3. Drag the widget to your panel or desktop.

3. Configure KWallet & API

  1. Right-click the widget icon and select Configure Kai Chat…
  2. Select a provider (e.g., Anthropic or OpenRouter) and enter your API Key.
  3. Click Save to KWallet. The model list will automatically sync.
  4. Select your model, apply changes, and start chatting!
