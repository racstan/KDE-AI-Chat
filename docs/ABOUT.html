Kai Chat

KDE Plasma AI widget docs and troubleshooting.

Author: RACHIT ASTHANA

Features

  • Centered chat header with top-left history toggle
  • Collapsible chat history sidebar
  • Per-chat rename and delete controls in history
  • Last modified timestamp per chat session
  • Per-message timestamp and assistant model label
  • Provider-specific settings in Configure only

KWallet Setup

Default wallet name is KaiChatWallet.

  1. Open KDE Wallet Manager.
  2. Create or unlock KaiChatWallet.
  3. Use Configure in Kai Chat to Save/Load provider keys.

Issue Check Commands

# List wallets
qdbus6 org.kde.kwalletd6 /modules/kwalletd6 org.kde.KWallet.wallets

# List installed plasmoids
kpackagetool6 --type Plasma/Applet --list | grep org.kde.plasma.kaichat

# Reinstall widget
cd /path/to/rachitkdeaichat
./install.sh
systemctl --user restart plasma-plasmashell.service

# QML lint
qmllint org.kde.plasma.kaichat/contents/ui/main.qml
qmllint org.kde.plasma.kaichat/contents/ui/ConfigGeneral.qml
